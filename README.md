# RelaxQL

A non-leaky nested component query library suitable for React components.
Inspired by [Facebook's
GraphQL](https://www.youtube.com/watch?v=9sc8Pyc51uU#t=6m45s).

## Nomenclature

Relay + Flux + GraphQL = RelaxQL
